# African Bites Restaurant Website

A modern, responsive restaurant website showcasing authentic African cuisine.

## Features

- **Fully Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Scrolling Navigation** - Easy navigation between sections
- **Interactive Menu** - Hover effects on food categories with overlay animations
- **Mobile-Friendly** - Hamburger menu for mobile devices
- **Contact Form** - Customer inquiry section

## Sections

1. **Home/Hero** - Eye-catching header with call-to-action
2. **About Us** - Restaurant story and values
3. **Food Categories** - Fresh fruits, vegetables, and grains
4. **Menu** - Full menu with prices in Kenyan Shillings (Ksh)
5. **Testimonials** - Customer reviews with star ratings
6. **Contact** - Contact form for customer inquiries

## Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling with animations and transitions
- **JavaScript/jQuery** - Smooth scrolling functionality
- **Font Awesome** - Icons for star ratings
- **Google Fonts** - Poppins font family

## Color Scheme

- Primary Color: `#d4621f` (Warm Orange)
- Hover Color: `#a84e18` (Dark Orange)
- Background: White and light gray tones
